Project Report

In this project, I created a simple web page to display student information and grades. The main goal was to provide a clear and organized way to present student records, including their IDs, names, courses, marks, and grades.

I started by structuring the HTML document with a header, a main content body, and a footer. The header includes a title and an image, which adds a visual element to the page. The main section features a table that lists the students' details. I used a <table> element to organize the data neatly, making it easy to read. Each row in the table represents a different student, and I included a footer to summarize the total number of students.

To enhance the user experience, I added a form for adding new student information. This form includes various input fields such as name, student ID, age, class, courses, expected grades, and more. There is a hover for each tb, tr and so on to make it looked cool. I made sure to include labels for each input to guide users on what information to provide. The form is styled to be visually appealing and user-friendly.I have apply different css style on button ,label etc.

For styling, I used CSS to create a reliable look throughout the page. I applied background colors, adjusted font sizes, and added hover effects to make the table interactive. The form is designed to stand out with a different background color and rounded corners, making it easy for users to focus on it.Styling i have used padding,margin, border etc to make it more reliable.

Overall, the project aimed to create a functional and visually appealing student information system that allows for easy viewing and input of student data. The combination of HTML for structure and CSS for styling ensures that the page is both informative and engaging.That make the page more beautiful
